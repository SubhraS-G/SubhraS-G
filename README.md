## 🎮 Projects

### 🔥 [SuGa BLAZE](https://github.com/SubhraS-G/SuGa-BLAZE) &nbsp;—&nbsp; *3D Endless Runner*

> Built from zero Unity 3D experience to a complete shipped game — in **22 days**.

Sprint through an ever-accelerating world, switch lanes, dodge obstacles, and collect coins for as long as you can survive.

| | |
|---|---|
| 🧍 Animated 3D character | run, jump & crouch |
| 🛣️ 3-lane switching system | fluid left/right movement |
| 🧱 Obstacle variety | walls, barriers & lane blockers |
| 🪙 Coin collection | multiple heights |
| ⚡ Progressive speed | gets faster over time |
| 🔊 Full audio | music + SFX |
| 🏆 Score persistence | best score saved between sessions |
